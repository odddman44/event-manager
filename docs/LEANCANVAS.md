# 린 캔버스 — 이벤트 매니저

---

## 1. 문제

> 가장 중요한 문제 1–3가지

1. **카카오톡 참여자 관리의 비효율** — 단체방에서 텍스트로 참여 의사를 수집하면 집계가 수작업이고 변경사항 추적이 불가능하다
2. **참여자 명단 파악 불가** — 누가 오는지, 몇 명인지를 한눈에 볼 수 없다
3. **참여 의사 변경 반영 어려움** — 취소·변경 메시지가 대화방에 묻혀 실제 명단과 불일치가 생긴다

### 대안 (현재 해결 방식)

- 카카오톡 단체방에서 손 댓글로 집계
- 구글 폼 / 구글 시트
- 직접 전화·문자 확인

---

## 2. 고객 세그먼트

> 타깃 고객을 나열하라

- **주최자**: 수영·헬스·동호회·친구 모임 등 소규모 이벤트를 반복적으로 여는 개인
- **참여자**: 회원가입 없이 빠르게 참여 의사를 전달하고 싶은 일반인

### 얼리 어댑터

- 주 1–2회 이상 소규모 이벤트를 운영하는 동호회장·모임 주최자
- 카카오톡 참여 집계에 시간을 쓰고 있어 불편함을 직접 느끼는 사람

---

## 3. 가치 제안 (UVP)

> 무심코 방문한 사람들을 잠재고객으로 바꾸는 단 하나의 메시지

**"링크 하나로, 회원가입 없이, 바로 참여 — 카카오톡 참여자 관리의 번거로움을 끝냅니다"**

### 상위 개념

소규모 모임을 위한 **"즉시 참여 링크"** 서비스

---

## 4. 솔루션

> 각 문제에 대해 가능한 솔루션의 개요

| 문제           | 솔루션                                                  |
| -------------- | ------------------------------------------------------- |
| 수작업 집계    | 이벤트 생성 → share_token 링크 → 참여자 목록 자동 집계  |
| 명단 파악 불가 | 이벤트 관리 페이지에서 참여자 목록·현황 실시간 확인     |
| 변경 추적 불가 | guest_token 기반 재방문 인식 → 메모 수정·참여 취소 지원 |

---

## 5. 채널

> 고객 도달 경로

- **카카오톡** — 공유 링크를 단체방·개인 채팅에 붙여넣기 (핵심 채널)
- **인스타그램 DM / 스토리** — 링크 공유
- **SMS / 이메일** — 링크 발송
- **구전** — 기존 주최자의 자연 추천

---

## 6. 수익 흐름

> 매출원을 나열하라

| 단계     | 수익 모델                                                          |
| -------- | ------------------------------------------------------------------ |
| MVP      | **무료** (사용자 확보 최우선)                                      |
| Post-MVP | 프리미엄 플랜 — 이벤트 무제한 생성, 고급 통계, 정원 초과 대기열 등 |
| 장기     | B2B 소규모 기업 행사·스포츠센터 구독 요금제                        |

## 7. 비용 구조

> 고정비와 변동비를 나열하라

**고정비**

- Supabase (DB·Auth) — 무료 티어 → 성장 시 유료 전환
- Vercel 배포 — 무료 티어 → 트래픽 증가 시 유료
- 도메인 비용 — 연 ~20달러

**변동비**

- 개발 인건비 (현재: 1인 개발)
- 마케팅·사용자 획득 비용

---

## 8. 핵심 지표

> 사업 현황을 알려주는 핵심 숫자들을 나열하라

- **이벤트 생성 수** — 주최자 활성도 측정
- **참여 신청 완료율** — 링크 클릭 대비 실제 참여 신청 비율
- **주최자 재방문율** — 두 번째 이벤트를 생성하는 비율 (제품 가치 검증)
- **링크 → 참여 전환율** — 공유 링크 클릭 후 참여 완료 비율
- **월간 활성 주최자(MAO)** — 성장 지표

---

## 9. 경쟁우위

> 다른 제품이 쉽게 흉내낼 수 없는 특징

- **참여자 회원가입 불필요** — 마찰을 극단적으로 낮춰 카카오톡 대비 참여 전환율 극대화
- **카카오톡 공유 플로우에 최적화된 UX** — 링크 클릭 → 이름 입력 → 완료, 3단계 이내
- **모바일 퍼스트 싱글 페이지** — 스크롤 없이 한 화면에서 참여 완료
- **guest_token 재방문 인식** — 앱 설치 없이도 참여 변경·취소 가능
